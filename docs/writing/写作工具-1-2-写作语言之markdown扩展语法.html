第一部分: 扩展语法

基本 Markdown 语法无法满足所有需求，扩展语法应运而生。扩展语法通过添加表格、代码块、脚注等元素来丰富 Markdown 的功能。

（一）表格

使用|分隔列，使用 ---- 定义表头和内容的分隔线

使用 : 来设置内容对齐方式，如: :---- 表示左对齐， ----: 表示右对齐， :----:表示居中对齐

⚠️ 提示：

  1. 表格中设置文本格式；如：加粗，斜体，行内代码

  2. 表格中可以使用HTML标签

  • 标题对齐示例:

| 左对齐表头 | 居中对齐表头 | 右对齐表头 | |:------ |:------:|-------:| | 1 | 2 | 3 | | 4 | 5 | 6 |

渲染效果如下:

左对齐表头

居中对齐表头

右对齐表头

1

2

3

4

5

6

  • 表体使用特殊符号示例:

| Markdown语法 | 渲染效果 | |:-------------:|:---------------:| | `# 一级标题` | <h1> 一级标题 </h1> | | `## 二级标题` | <h2> 二级标题 </h2> | | `### 三级标题` | <h3> 三级标题 </h3> | | `#### 四级标题` | <h4> 四级标题 </h4> | | `##### 五级标题` | <h5> 五级标题 </h5> | | `###### 六级标题` | <h6> 六级标题 </h6> |

（二）代码块

  • 行内代码 ：使用反引号 ` 包裹代码。

python的输出结果如下：`print("Hello, World!")`

渲染效果如下:

python的输出结果如下： print("Hello, World!")

  • 围栏代码块

使用三个反引号 ``` 创建代码块， 在反引号后指定代码语言，可实现语法高亮。

``` json { "firstName": "John", "lastName": "Smith", "age": 25 } ```

渲染效果如下:

{ "firstName": "John", "lastName": "Smith", "age": 25 }

（三）脚注

⚠️ 警告： 不同平台对脚注的支持程度不同，使用前请确认目标平台是否支持

脚注与链接的区别如下所示：

链接：[文字](链接) 脚注：[文字](脚注解释 "脚注名字")

有人认为在大前端时代 的背景下，移动端开发（Android、IOS）将逐步退出历史舞台。

全栈工程师在业务开发流程中起到了至关重要的作用。

脚注内容请拉到最下面观看。

（四） 删除线

请在单词前后使用两个波浪号 ~~。

~~世界是平坦的。~~ 我们现在知道世界是圆的。

渲染效果如下:

世界是平坦的。 我们现在知道世界是圆的。

（五）任务列表

任务列表语法：使用 [ ] 表示未完成， [x] 表示已完成。

- [ ] 未完成的任务 - [x] 已完成的任务（打勾）

渲染效果如下:

  • [ ] 未完成的任务

  • [x] 已完成的任务（打勾）

第二部分: 变通语法

我们除了能使用扩展语法，还可以使用 HTML 变通方法实现特殊效果。

⚠️ 注意：HTML标签兼容性因平台而异

（一） 下划线

一些文字 <ins>将被加下划线</ins>。

渲染效果如下:

一些文字 。

（二） 缩进（Tab）

&nbsp;&nbsp;&nbsp;&nbsp;这是一个缩进的段落。

渲染效果如下:

这是一个缩进的段落。

（三） 文字居中

<p style="text-align:center">这段文字居中显示。</p>

渲染效果如下:

这段文字居中显示。

（四） 文字颜色

<font color="red">这段文字是红色的！</font> <p style="color:blue">这段文字是蓝色的。</p>

渲染效果如下:

这段文字是红色的！

这段文字是蓝色的。

（五） 隐藏注释

这是可见的段落。 [这是一个隐藏的注释]: # 这是另一个可见的段落。

渲染效果如下:

这是可见的段落。

这是另一个可见的段落。
