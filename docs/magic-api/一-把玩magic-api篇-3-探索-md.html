三、接口配置

接口分组

友情提示：

  • 分组路径：

    • 路径格式：/分组1/分组2/.../分组n

  • 接口分组可以嵌套创建

推荐分组设计:

  • 基础分组：

    • 路径：/base

    • 说明：基础分组，包含系统的基础数据接口，如：用户、角色、权限、字典等

  • 业务分组：

    • 路径：/business

    • 说明：业务分组，包含系统的业务数据接口，如：订单、商品、库存等

  • 扩展分组：

    • 路径：/extension

    • 说明：扩展分组，包含系统的扩展数据接口，如：短信、支付、物流等
