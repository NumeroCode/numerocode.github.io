二、域名是啥玩意儿？不就是个网址吗？

域名（Domain Name）说白了就是网站的"门牌号"，是互联网上某个服务器或网络系统的名字。

举个例子：

  • baidu.com - 百度的域名

  • github.com - GitHub的域名

  • shushuobiancheng.com - 哎呀，这个好像还没被注册（如果真有这个域名那纯属巧合😄）

域名的结构一般是这样的：

www.example.com │ │ │ │ │ └─ 顶级域名(.com, .net, .org等) │ └─ 二级域名(可以自定义) └─ 子域名(通常为www，也可以是blog、wiki等)

对于个人博客来说，个人觉得选个简短好记的.com域名就足够了。 要求不高的图便宜的也可以选其他后缀的域名，比如：.net、.xyz、.me、.io、.top、.vip、.site

tip: 域名也可以作为一种资产哟，万一哪天自己的域名被哪个公司看上了，他可能会花高价钱购你的域名

三、域名申请流程：其实比你想的简单多了

别被"申请"这个词吓到，现在申请域名比买菜还简单。

1. 选择域名服务商

主流的有：

  • 阿里云 ：国内老大，服务稳定

  • 腾讯云 ：界面友好，活动多

  • 华为云 ：技术实力强，价格实惠

  • 国外的 ：Namecheap、Godaddy等（需要备案的话就别考虑了）

个人推荐腾讯云或者阿里云，界面友好，文档齐全。

2. 域名查询和选择

进入域名购买页面，输入你想注册的域名，系统会告诉你是否被占用。

选域名的小技巧：

  • 简短好记（最好是你的名字或昵称）

  • 避免特殊字符和连字符

  • .com后缀优先

  • 避免容易拼错的词汇

3. 购买和支付

选好域名后直接加入购物车，付款就行。一年一般几十块，不贵。

4. 实名认证

国内服务商都需要实名认证，上传身份证就行。

四、域名解析/使用：让你的域名真正"工作"起来

申请域名只是第一步，还得让它指向你的博客系统才行。

域名解析基本概念

域名解析就是把你的域名指向服务器IP地址的过程。

比如：

shushuobiancheng.com → 123.456.789.123 (服务器IP)

解析设置步骤

  1. 登录域名管理后台

  2. 找到"域名解析"或"DNS管理"

  3. 添加解析记录：

    • A记录 ：直接指向服务器IP

    • CNAME记录 ：指向另一个域名（比如GitHub Pages）

  4. 等待生效（一般几分钟到几小时）

常见使用场景

  1. 指向云服务器 ：直接解析到你的VPS IP

  2. 指向GitHub Pages ：CNAME解析到username.github.io

  3. 指向博客托管平台 ：根据平台要求设置解析
